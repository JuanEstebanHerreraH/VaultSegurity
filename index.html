    Bot de WhatsApp

    Estado del Bot

    Escanea el código QR con WhatsApp para vincular el bot a tu cuenta.

    📱 Escanear QR

    Presiona "Conectar Bot" para generar el código QR.

    Luego ábrelo en WhatsApp → Dispositivos Vinculados → Vincular Dispositivo

    ⚡ Cómo Activar el Bot

    1
    Instala las dependencias del bot: abre una terminal en la carpeta del proyecto y ejecuta npm install whatsapp-web.js qrcode
    2
    Presiona "Conectar Bot" en este panel para que aparezca el código QR.
    3
    Abre WhatsApp en tu teléfono → toca los tres puntos ⋮ → Dispositivos Vinculados → Vincular Dispositivo
    4
    Escanea el QR con tu cámara. El bot quedará vinculado a tu número de WhatsApp.
    5
    Envíate un mensaje a ti mismo desde WhatsApp usando los comandos de abajo.

    💬 Comandos Disponibles

    !archivos
    1. Escribe !archivos → el bot responde con la lista de nombres de archivos guardados en tu bóveda.
    2. Escribe !ver archivos para ver el detalle completo.
    3. Escribe !eliminar archivo [nombre] para mover uno a la papelera.
    !imagenes
    1. Escribe !imagenes → lista de nombres de imágenes en tus álbumes.
    2. Escribe !ver imagenes para ver detalles.
    3. Escribe !eliminar imagen [nombre] para moverla a la papelera.
    !notas
    1. Escribe !notas → lista de títulos de notas guardadas.
    2. Escribe !ver notas para ver el contenido de cada nota.
    3. Escribe !eliminar nota [título] para borrarla.
    !eventos
    1. Escribe !eventos → lista de próximos eventos del calendario.
    2. Escribe !ver evento [título] para ver sus detalles.
    3. Escribe !eliminar evento [título] para borrarlo.
    !ayuda
    El bot te responde con todos los comandos disponibles.

    ⚠️ Nota Importante

    El bot de WhatsApp requiere que la app VaultSecurity esté abierta y desbloqueada en tu computadora para funcionar. No opera de forma remota si la app está cerrada.

    La conexión QR puede necesitar renovarse cada ci La conexión QR puede necesitar renovarse cada ciertos días por seguridad de WhatsApp.
